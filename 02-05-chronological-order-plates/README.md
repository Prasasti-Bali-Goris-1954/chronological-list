# Overview

A digitised searchable [table](https://github.com/Prasasti-Bali-Goris-1954/chronological-list/blob/main/02-05-chronological-order-plates/02-05-chronological-order-plates-tbl.tsv) of list of Old Balinese inscriptions published in Goris (1954: 343-347, Band II):

> Goris, Roelof. 1954. Prasasti Bali I & II. Bandung: Masa Baru. https://archive.org/details/r-goris-prasasti-bali 

[Gede Primahadi W. Rajeg](https://github.com/gederajeg) xml-tagged [the OCR file](https://github.com/Prasasti-Bali-Goris-1954/chronological-list/blob/main/02-05-chronological-order-plates/02-05-chronological-order-plates.xml) for the table and designed an [R script](https://github.com/Prasasti-Bali-Goris-1954/chronological-list/blob/main/02-05-chronological-order-plates/02-05-chronological-order-plates-code.R) to parse the XMLs and turn them into a table.
